# ABstickman
这个项目使用数据包和资源包还原了ab柴里面的各种东西
## 方块权杖
方块权杖使用数据包和资源包为游戏添加了**便携式发射器**和**权杖**两个物品
使用/function ab:item/rod以获取权杖
## 权杖
手持权杖时，可以蹲下右键将副手上的方块装填上去，可以被装填上去的方块有：
命令方块，钻石块，下界合金块，黑曜石，基岩，铁砧，屏障，蜂巢，书架，酿造台，营火，炼药锅，圆石，滴水石块，绿宝石块，附魔台，末地传送门框架，金块,砂轮，冰块，铁块，青金石块，避雷针，岩浆块，下界岩，音符盒，橡木原木，玩家的头，雪块，红石块，TNT,羊毛，凋零骷髅头颅，幽匿感测体，便携式发射器。
### 命令方块权杖
手持按左键打开指令列表，列表自带三个指令，分别是自定义指令，tp最近的玩家，kill最近的玩家。
右键使用选择的指令。
使用自定义指令后，玩家的副手将会替换为一个输入工具（实际为书与笔），空手打开输入工具后，可在其中写入指令，成书名字可写入指令的名字。成书后，再次主手持命令方块权杖，副手持输入工具，可将刚刚成书的指令记录到列表。
### 钻石块权杖
基础伤害+5，左键时将会破坏前方一定范围的方块
### 下届合金块权杖
基础伤害+10，左键时将会破坏前方巨大范围的方块
### 黑曜石权杖
左键时将会推动前方一定范围的方块
### 基岩权杖
基础伤害+40，击退+20，左键时将会推动前方一定范围的方块，右键可以持续召唤临时基岩攻击，
### 铁砧权杖
基础伤害+20，攻击速度-2，速度-999，重力+1
### 屏障权杖
左键发射屏障流，右键召唤临时屏障墙
### 蜂巢权杖
左键发射蜜蜂攻击，蜜蜂攻击带毒，有概率召唤会爆炸的苦力怕蜜蜂。右键可以标点，控制蜜蜂飞向标点位置
### 书架权杖
右键喷出书与纸散射攻击，纸的数量是书的两倍，书的伤害是纸的两倍
### 酿造台权杖
左键喷出四种喷溅药水，分别是缓慢，瞬间伤害，中毒，虚弱。右键持续喷出多种药水粒子，分别是缓慢，瞬间伤害，中毒，虚弱，虫蚀，蓄风，渗浆，盘丝
### 营火权杖
左键持续性喷出火焰，右键向下喷出火焰飞行
### 炼药锅权杖
右键吸收酿造台的药水粒子
### 圆石权杖
左键在前方放置一次圆石制方块，右键在前方持续放置圆石制方块
### 滴水石块权杖
左键在前方召唤一个从下至上刺出的滴水石锥，右键持续在前方召唤一条从下至上刺出的滴水石锥
### 绿宝石块权杖
基础伤害+7，左键消除前方一次魔法相关的其他权杖放出的弹射物，右键持续消除前方魔法相关的其他权杖放出的弹射物
### 附魔台权杖
左键给前方生物手中物品附魔，右键给自己身上及另一只手上的物品附魔，附魔不限物品仅限的附魔
### 末地传送门框架权杖
左键在远处召唤传送门，右键在自己旁边召唤传送门。每两次召唤的传送门相通
### 金块权杖
基础伤害+4，击退+3
### 砂轮权杖
右键持续攻击前方生物，无视无敌帧。若生物在方块内，则额外造成一次伤害
### 冰块权杖
左键召唤三道冰块流向前飞行直到碰到方块或生物，碰到生物将会冰封该生物。右键持续召唤冰块流
### 铁块权杖
基础伤害+3，右键向前方发射锁链，被锁链命中的目标将会获得缓慢效果
### 青金石块权杖
手持时将会漂浮起来，蹲下可以停止漂浮。左键附魔一个方块，将其变为实体并出现附魔光效，再次左键将其复原。蹲下左键放出青金石激光，造成魔法伤害。方块保留其内容物。右键控制附魔的方块向前方飞行，但y轴的方向相反
### 避雷针权杖
左键在前方召唤闪电，右键持续在前方召唤闪电
### 岩浆块权杖
左键攻击使受攻击实体着火，右键向前方召唤火球
### 下界岩权杖
左键召唤三道下界岩流向前飞行，直到碰到基岩或生物，碰到生物将会对其造成伤害。右键持续召唤下界岩流
### 橡木原木权杖
左键在前方放置一次橡木制方块，右键在前方持续放置橡木制方块
### 玩家的头权杖
无作用，当手持时被雷劈将会变为Herobrine的头权杖
### Herobrine的头权杖
手持时飞行，蹲下时可以停止飞行。免疫摔落伤害，鞘翅冲击伤害。蹲下左键可以瞬移到视角前方方向。右键持续放出herobrine的头，造成out_of_world伤害
### 雪块权杖
左键发射一个雪球，右键持续喷射雪球
### 红石块权杖
左键发射三枚小激光弹，右键召唤激光，均造成火焰伤害
### TNT权杖
左键发射一次tnt，右键持续向前发射tnt
### 羊毛权杖
左键放置一次羊毛，右键持续放置羊毛
### 凋零骷髅头颅权杖
左键发射一次凋灵之首，右键持续发射凋灵之首，有概率发射蓝色凋灵之首
### 幽匿感测体权杖
手持时持续增加闪避次数，受到伤害时无敌，并消耗一次闪避次数向前闪避。当有弹射物在周围时，也会消耗一次闪避次数。闪避后增加下一次增加闪避次数的时间
### 发射器权杖
无限发射便携式发射器种的物品，左键发射一次，右键持续发射
## 便携式发射器
由一个普通发射器合成而来。可以向里面如同收纳袋一样装填物品
左键随机发射其中弹射物
